# DVGateway — 한 페이지 요약

> **AI를 회사 전화에 연결하는 가장 빠른 방법.**
> 기존 사무실 전화·PBX·번호를 그대로 유지하면서, AI 음성 상담·자동 안내·회의록 자동화를 며칠 안에 도입합니다.

---

## 🔑 핵심 가치 (Why)

- ⏰ **24/7 응대** — 점심·야간·주말에 놓치던 전화를 0건으로
- 💸 **인건비 절감** — 상담원 1명당 월 250만원 → 시스템 월 3~5만원
- 🚀 **5분 도입** — 코딩·앱 설치 없이 기존 전화선 그대로 연결
- 🇰🇷 **한국 환경 최적화** — 한국어 STT/TTS·사투리·국내 PBX 검증
- 🔒 **온프레미스 가능** — 음성·텍스트가 외부에 안 나가게 사내 설치

| 시나리오 | 누가 쓰나 | 월 예상 비용* | 절감 효과 |
|----------|-----------|--------------|-----------|
| **AI 전화 상담원** | 의원·카페·쇼핑몰·부동산 | 약 3만원 | 직원 1명 채용비 250만원/월 절약 |
| **무인 안내 IVR** (코딩 0) | 모든 사업장 | 거의 0원 | 일반 통화로 응대하던 직원 시간 100% 회수 |
| **회의록 자동 생성** | 임원진·법무·연구개발 | 회의 1시간당 500원 | 1회당 1~2시간 정리 시간 → 자동 |
| **콜센터 실시간 어시스턴트** | 상담원 다수 운영 조직 | 상담원당 월 4.4만원 | 신입 교육 3개월 → 2주 |
| **회의 YouTube 라이브** | 주주총회·웨비나 | 1시간당 500원 | 방송업체 300만원 → 사내 무인 운영 |

## ⚖️ 전통 방식 vs DVGateway

| 항목 | 전통 ARS 시스템 | DVGateway |
|------|-----------------|-----------|
| 안내 멘트 수정 | 외주 의뢰 (수일~수주) | 코드/설정 1줄, 5초 반영 |
| 대화형 응대 | 불가능 ("1번을 누르세요"만) | AI가 자연어로 대화 |
| 다국어 | 언어별 멘트 별도 제작 | 다국적 STT/TTS 한 시스템 |
| 도입 비용 | 수천만~수억원 | 라이선스 + 클라우드 사용량 |
| 회의록·녹음 분석 | 별도 솔루션 | 동일 시스템에 내장 |
| YouTube 라이브 송출 | 방송 장비 + 인력 | 자동 (전화 회의가 곧 라이브) |
| 동시통화 확장 | 하드웨어 증설 | 라이선스 티어 변경만 |

## 🚀 도입 4단계

1. **30분 — 진단 통화** : 영업팀이 현재 PBX·통화량·목적을 확인하고 적합도 평가
2. **1일 — 설치** : 사내 서버 또는 클라우드에 원라인 명령으로 설치 (`curl ... | bash`)
3. **2~5일 — 시나리오 구성** : 안내 멘트·답변 매뉴얼·라우팅 규칙 정의
4. **1주 — 파일럿 → 정착** : 일부 회선부터 시작해 점진 전환

## 🧩 무엇이 다른가 (기술 차별점)

- ✅ **Asterisk PBX 표준 호환** — 신규 통신 인프라 도입 불필요
- ✅ **STT/LLM/TTS 자유 선택** — Deepgram·Claude·OpenAI·ElevenLabs·자체 GPU 등 30+개 어댑터
- ✅ **멀티테넌트 SaaS** — 한 서버로 여러 고객사 격리 운영
- ✅ **6.종 통화 프로파일** — 풀 AI / 모니터링 / 무인 IVR(lite) / VoiceFlow / 컨퍼런스 / Warm Transfer
- ✅ **라이선스 한 번에 명확** — 동시통화 수 기준, 외부 API 호출에 별도 마진 없음
