# 가이드 5 — 회의를 YouTube로 라이브 송출

> "주주총회·웨비나·교육 세미나를 외부에도 보여주고 싶은데, 비싼 방송 장비 없이 가능할까?" 전화로 진행되는 회의가 그대로 YouTube 라이브로 송출됩니다.

## 🎯 이 가이드로 만들 수 있는 것

- **전화 회의를 YouTube 라이브로 실시간 송출** — 시청자는 영상·음성 자유 시청
- **자동 자막** — 발언 내용이 한국어/영어 자막으로 실시간 표시
- **참여자 이름 화면 표시** — "지금 발언 중: 김부장 (영업본부장)"
- **회사 로고·배경 이미지** — 전문 방송처럼 보이는 화면
- **공지 티커** — 화면 하단에 "다음 주 정기 휴무" 같은 자막 흐름
- **회의 종료 후 다시보기 자동 저장** — YouTube에 자동 아카이브

## 💼 이런 분들께 추천

- **상장사·중견기업** — 주주총회·실적발표 라이브
- **교육·학원** — 온라인 강의, 외부 강사 초청 세미나
- **종교·문화 단체** — 예배·강연 실시간 송출
- **컨퍼런스·웨비나 운영자** — 외부 강사를 전화로 모셔도 라이브 가능
- **공공기관** — 시민 공청회·설명회

## 💰 예상 비용

| 항목 | 비용 |
|------|------|
| YouTube 라이브 송출 | **무료** ⭐ (YouTube 자체) |
| 자막용 음성 인식 | 시간당 약 300원 |
| 자동 화자 표시 | 시간당 약 200원 |
| 인터넷 회선 추가 비용 | 기존 사무실 회선이면 보통 추가 없음 |
| **합계 (1시간 라이브 기준)** | **약 500원** |

> 비교: 전문 방송 장비 + 인력 = 회당 200~500만원.
> 1년에 12회 분기별 라이브 진행 시 시스템 비용은 **연간 6,000원**, 절감액은 **수천만원**.

## 📋 도입 절차

### 1단계 — YouTube 채널 준비 (1시간)
- 일반 YouTube 계정으로 가능 (구독자 0명도 OK)
- "라이브 스트림 사용" 활성화 (24시간 대기 시간 있음)
- 스트림 키 발급 받기

### 2단계 — 화면 디자인 결정 (1~2일)
다음 자료를 준비합니다:
- 회사 로고 (PNG, 투명 배경 권장)
- 배경 이미지 (1920x1080, 회사 분위기에 맞게)
- 한글 폰트 파일 (Noto Sans CJK 등 무료 폰트 가능)
- 티커에 흐를 공지 문구 (선택)

### 3단계 — 시스템 설정 (개발자 1시간)
- YouTube 스트림 키를 게이트웨이에 등록
- 자동 송출 ON: 회의실에 누군가 들어오는 순간 자동으로 YouTube 라이브 시작
- 종료 조건: 마지막 사람이 나가면 자동 종료 + YouTube 아카이브 저장

### 4단계 — 시범 송출 (30분)
- 사내 회의실 번호로 2~3명이 전화
- YouTube 라이브 페이지에서 자막·화자·로고 확인
- 부족한 부분은 2단계 자료 보완

### 5단계 — 본격 운영
- 매번 라이브 키를 재설정할 필요 없음 — 자동 반복

## 🌟 활용 사례

**중견 IT기업 분기 실적발표** (가상)
> 매분기 실적발표를 위해 방송 업체 섭외 (회당 300만원).
> 도입 후: 사내 회의실 5500번으로 임원진이 전화 → 자동 YouTube 라이브.
> 외부 투자자·기자가 채널에서 실시간 시청 + Q&A 채팅 가능.
> 연 1,200만원 절감, 사내 운영 부담 최소.

**대형 교회 주일예배** (가상)
> 코로나 이후 온라인 예배 수요 지속.
> 강대상 마이크 음성이 자동으로 YouTube 라이브로 송출.
> 화자가 누구인지(목사·찬양인도자) 자동 화면 표시.
> 거동이 불편한 성도들도 실시간 참여 가능.

**학원·평생교육원 강의** (가상)
> 유명 강사 1회 특강을 평일 저녁에 진행.
> 강사실에서 전화 연결만으로 라이브 송출.
> 수강생은 YouTube 링크로 어디서나 시청.
> 강의 후 자동 저장된 영상은 다음 기수에 재활용.

## 📺 자동 자막 vs 수동 자막

| 항목 | 자동 자막 (본 시스템) | 수동 자막 (속기사) |
|------|----------------------|--------------------|
| 정확도 | 90~95% | 99%+ |
| 지연 시간 | 1~2초 | 3~5초 |
| 비용 | 시간당 300원 | 시간당 5~10만원 |
| 외국어 동시 지원 | 가능 (한·영 동시) | 별도 인력 필요 |
| 24시간 운영 | 가능 | 어려움 |

> 법적 효력이 필요한 공식 발표(예: 상장사 실적발표)는 자동 자막 + 사후 수동 보정 병행 권장.

## 🛠️ 화면 커스터마이징 옵션

다음 요소를 자유롭게 조합할 수 있습니다.

- **타이틀**: "2026년 정기 주주총회"
- **로고 위치**: 좌상단/우상단/하단 가운데
- **자막 스타일**: 폰트 크기·색·배경·테두리
- **참여자 표시**: 자동/수동/숨김
- **티커 속도**: 천천히/보통/빠르게
- **배경**: 단색/이미지/그라데이션

> 모두 화면 출력 후 즉시 확인 가능. 디자이너 없이 사장님 취향대로 조정.

## 🔒 비공개 라이브 옵션

- **공개**: 누구나 시청 (URL 알면 접근)
- **일부 공개**: URL 아는 사람만 (검색·추천 노출 없음) — 가장 흔함
- **비공개**: 초대된 계정만 시청 가능 (가족 공유 등)

> 주주총회·내부 교육 등은 "일부 공개"로 운영하면 외부 노출 없이 안전.

## ⚖️ 다른 가이드와의 차이

| 가이드 | 주 용도 | 시청자 |
|--------|--------|--------|
| #3 회의록 자동 생성 | **사후** 텍스트 기록 | 회의 참여자·임원진 |
| #5 YouTube 라이브 (본 문서) | **실시간** 영상 송출 | 외부 일반 시청자 |

→ 두 기능을 **함께** 쓰면 라이브 송출하면서 텍스트 회의록도 동시 생성됩니다.

## ⚠️ 주의사항

- **인터넷 상향 대역폭**: 라이브 영상 송출은 업로드 속도가 5Mbps 이상 권장
- **회의 참여자 동의**: 외부 송출 사실을 모든 참여자에게 사전 안내
- **개인정보 보호**: 자막에 개인정보 포함 가능 → 자동 마스킹 권장
- **저작권**: 회의 중 음악·영상 재생 시 YouTube가 자동 차단할 수 있음
