# 가이드 1 — AI 전화 상담원 5분 만들기

> "전화는 자꾸 놓치는데 사람을 더 뽑긴 어렵다." 24시간 일하는 AI 전화 상담원이 해결책이 될 수 있습니다.

## 🎯 이 가이드로 만들 수 있는 것

- **24시간 자동 응답** — 점심시간·야간·주말에도 전화를 놓치지 않습니다
- **자연스러운 대화** — 단순 ARS와 달리 사람처럼 대화합니다
- **고객 정보 자동 기록** — 통화 내용을 텍스트로 저장 (회의록 기능 포함)
- **상담원 연결** — AI로 해결 안 되는 건만 사람에게 전달

## 💼 이런 분들께 추천

- **소규모 의원/한의원** — 예약 문의, 진료 시간 안내
- **카페·식당** — 영업시간·메뉴·예약 문의
- **부동산 중개소** — 매물 문의, 첫 상담 자동화
- **온라인 쇼핑몰** — 배송·환불 문의 1차 대응
- **법무사·세무사 사무소** — 상담 일정 잡기, 일반 안내

## 💰 예상 비용 (월 1,000통 통화 기준)

| 항목 | 비용 |
|------|------|
| DVGateway 라이선스 (Starter 티어, 동시통화 1) | 월 무료 또는 협의 |
| 음성 인식 (Deepgram 등) | 약 5,000원 |
| AI 대화 (Claude / GPT) | 약 15,000원 |
| 음성 합성 (ElevenLabs 등) | 약 10,000원 |
| **합계** | **월 약 30,000원** |

> 사람 한 명을 안 뽑으면 월 250만원 이상 절감.
> 통화량이 늘어도 비용은 통화량에 비례해 선형 증가합니다.

## 📋 도입 절차 (실제 작업 시간 5분)

### 1단계 — 시나리오 정하기 (10분)
어떤 응답을 원하는지 한국어로 적습니다.

```
당신은 OO병원 안내 직원입니다.
환자에게 친절하게 응대하고, 다음을 안내하세요:
- 진료 시간: 평일 9-18시, 토요일 9-13시
- 위치: 서울시 강남구 OO로 123
- 예약은 카카오톡 채널 @OO병원 으로 안내
긴급 상황이면 119를 안내하세요.
```

### 2단계 — 음성 선택 (5분)
- 음성 합성(TTS) 서비스에서 회사 분위기에 맞는 목소리 선택
- ElevenLabs, Google Cloud TTS, OpenAI TTS 등 30종 이상 중 선택
- 본인 목소리 복제도 가능 (추가 비용)

### 3단계 — 봇 실행 (5분, 개발자 1인)
간단한 코드 한 번에 동작합니다.

```javascript
const gw = new DVGatewayClient({ baseUrl, auth: { apiKey } });
gw.pipeline()
  .stt(deepgramAdapter)
  .llm(claudeAdapter, { systemPrompt: '...' })  // 1단계 시나리오
  .tts(elevenLabsAdapter, { voice: '...' })     // 2단계 음성
  .start();
```

### 4단계 — 전화번호 연결 (통신사 작업)
- 기존 사무실 전화번호를 그대로 사용 가능
- 평일 야간/주말에만 AI로 자동 전환되도록 시간대별 라우팅도 가능

### 5단계 — 테스트 통화 1회
- 사무실 전화로 직접 전화 걸어 대화 확인
- 부족한 응답은 1단계 시나리오를 다듬어 즉시 반영 (재배포 5초)

## 🌟 활용 사례

**소아과의원 사례** (가상)
> 평일 17~22시 야간에 평균 30통씩 전화가 왔지만 모두 놓치고 있었음.
> AI 상담원 도입 후: "약 복용 시간 문의", "예약 변경" 같은 단순 문의는 95% 자동 응답.
> 응급 증상은 즉시 119 안내 또는 당직 의사 휴대폰으로 자동 전환.
> 다음 날 아침 통화 요약본을 이메일로 받아 후속 조치.

**부동산 중개소 사례** (가상)
> 인터넷 광고 클릭 후 전화 문의가 들어왔는데 외근 중이라 못 받음.
> AI 상담원이 매물 정보를 안내하고 관심 매물 ID를 메모.
> 다음 날 영업사원이 메모 보고 콜백 → 계약 성사율 40% 증가.

## ⚠️ AI가 잘 못하는 것 (현실적 한계)

- **복잡한 가격 협상** → 사람에게 자동 전환되도록 설정
- **법적 책임이 따르는 답변** → "전문 상담원에게 연결하겠습니다"로 회피
- **사투리·노인 발음** → 인식률 80~90%대 (지역 사투리는 70~80%)
- **여러 명이 동시에 말할 때** → 1:1 대화에 최적화

→ 도입 후 첫 2주는 통화 녹음·텍스트를 검토하면서 시나리오를 다듬는 것이 권장됩니다.
