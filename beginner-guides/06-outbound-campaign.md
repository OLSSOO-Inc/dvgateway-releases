# 가이드 6 — AI 자동 발신 (예약 리마인더·만족도 조사·캠페인)

> "예약 전날 전화로 확인하는 일이 직원 한 명의 하루 일과의 절반이다." 자동으로 전화를 걸어 안내·확인·설문까지 마치는 시스템입니다.

## 🎯 이 가이드로 만들 수 있는 것

- **예약 리마인더 자동 발신** — "내일 오후 2시 예약 확인 부탁드립니다. 1번 확인, 2번 변경"
- **만족도 조사 자동 콜** — 통화 종료 24시간 후 자동 전화, NPS 점수 수집
- **결제 안내·연체 알림** — "이번 달 결제일이 다가왔습니다…"
- **이벤트 안내 캠페인** — 특정 고객군 1만명에 동시 발신
- **본인 인증 콜백** — 신규 가입자에 자동으로 전화 걸어 인증코드 안내
- **노쇼 방지** — 약속 시간 1시간 전 자동 알림
- **모든 결과 자동 기록** — 누가 받았는지·뭐라 답했는지·녹음까지 CRM에 자동 저장

## 💼 이런 분들께 추천

- **병의원·치과·미용실** — 노쇼 방지가 매출과 직결
- **학원·교육기관** — 결석/지각 안내, 학부모 알림
- **금융·보험사** — 결제일·만기일·약관 변경 안내 (의무 안내사항)
- **이커머스** — 배송 지연·반품 진행 상황 알림
- **부동산·중개업** — 매물 등록 후 일정 시간 뒤 만족도 콜
- **B2B SaaS** — 트라이얼 종료 1주일 전 자동 안내
- **공공기관** — 검진 안내·예방접종·민원 처리 결과 통보

## 💰 예상 비용 (월 1,000건 발신 기준)

| 항목 | 비용 |
|------|------|
| 발신 통신비 (KT/SKT 등 국내 일반) | 약 30,000원 |
| 음성 합성 (TTS, 첫 1회만 합성 후 캐싱) | 거의 0원 |
| 음성 인식 (응답을 듣는다면) | 약 5,000원 |
| 시스템 라이선스 | 동시 발신 수에 따라 |
| **합계** | **월 약 4~5만원** |

> 직원 1명이 하루 50통씩 수동 발신 (월 1,000통) → 인건비 환산 약 250만원.
> **연간 약 3,000만원 절감** 가능.

## 🔥 인바운드와 어떻게 다른가

| 항목 | 인바운드 (가이드 #1~#5) | 아웃바운드 (이 가이드) |
|------|----------------------|----------------------|
| 누가 거나? | 고객 → 회사 | 회사 → 고객 |
| 시점 통제 | 고객 마음 | 회사 마음 (오전 9시-오후 6시 등) |
| 동시 발신 | 한 명씩 | 1만 명 동시 가능 |
| 응답률 | 100% (받는 사람만 통화) | 평균 40~60% (부재중·거절 포함) |
| 법적 규제 | 적음 | 광고성 발신은 동의 필요 |

→ **법적 주의**: 한국은 광고성 발신에 사전 동의(옵트인) 필수. 단, **계약 이행 관련 안내**(예약 확인·결제일 안내·서비스 종료 알림)는 동의 없이도 가능.

## 📋 도입 절차

### 1단계 — 발신 대상 데이터 준비 (1일)
엑셀 한 장이면 충분합니다.

```
| 전화번호       | 이름   | 예약일시            | 시술 종류 |
|----------------|--------|---------------------|-----------|
| 010-1234-5678  | 김환자 | 2026-05-21 14:00    | 정기검진  |
| 010-2345-6789  | 이고객 | 2026-05-21 15:30    | 스케일링  |
| ...            | ...    | ...                 | ...       |
```

→ CRM에서 자동으로 받아오게 연결할 수도 있음 (가이드 #7 참조).

### 2단계 — 안내 멘트 시나리오 작성 (30분)

```
[발신 멘트]
안녕하세요, {이름}님. OO치과입니다.
내일 {예약일시}에 {시술 종류} 예약이 잡혀 있습니다.
시간 그대로 진행하시려면 1번, 변경을 원하시면 2번,
취소를 원하시면 0번을 눌러주세요.

[1번 응답]
확인해 주셔서 감사합니다. 내일 뵙겠습니다.

[2번 응답]
변경 일정은 상담원이 다시 전화드리겠습니다.

[0번 응답]
예약이 취소되었습니다. 다시 방문하실 때 연락 주세요.

[부재중·미응답]
음성메시지: "내일 OO치과 예약 안내입니다. 확인 부탁드립니다."
```

### 3단계 — 발신 시간대 설정 (10분)
- 영업시간 내에만 (예: 평일 10시-17시)
- 휴대폰 야간 차단 시간 회피
- 대량 발신 시 분당 발신 수 제한 (통신사 정책)

### 4단계 — 파일럿 발신 50건 (1시간)
- 작은 그룹부터 시작해 응답률·통화 품질 확인
- 부적절한 멘트는 즉시 조정 (재시작 5초)

### 5단계 — 정기 운영
- CRM/DB에서 자동으로 발신 대상 수집 → 매일 정해진 시간에 자동 발신
- 결과는 자동으로 CRM에 기록 (응답 여부, 선택 번호, 통화 시각)

## 🌟 활용 사례

**중소형 치과의원** (가상, 일 평균 예약 30건)
> 노쇼율 도입 전 15% → 도입 후 4%.
> 매출 손실 환산 월 800만원 → 200만원으로 감소.
> 카운터 직원이 매일 오후 1~2시간 쓰던 확인 전화 업무가 0이 됨.

**온라인 학원** (가상, 수강생 5,000명)
> 결석 학생 학부모에게 자동 전화: "오늘 OO 강의에 결석했습니다. 1번 사유 등록, 2번 보강 신청".
> 이전: 강사·조교 30명이 매일 결석자에게 수동 전화 → 1인당 1시간 소요.
> 이후: 자동 발신, 강사는 수업 준비에 집중.

**렌탈 가전 회사** (가상)
> 정수기 필터 교체 주기 도래 시 자동 발신.
> "○○님 정수기 필터 교체 주기가 도래했습니다. 방문 예약을 잡으시려면 1번을…"
> 방문 매출 30% 상승.

**카드사** (가상)
> 카드 발급 24시간 후 자동 발신: "카드 무사히 받으셨나요?" + 본인 인증 코드 안내.
> 사기 거래 사전 차단율 상승, 고객은 카드사 신뢰도 ↑.

## ⚠️ 발신 전 체크리스트

- [ ] 발신 대상자 사전 동의 또는 계약 관련 안내 해당 여부 확인
- [ ] 발신 시간대 적절성 (밤·새벽 금지)
- [ ] "광고가 아니라 안내"임을 명확히 (시작 멘트에 명시)
- [ ] 수신 거부 안내 ("0번을 누르시면 향후 안내가 발송되지 않습니다")
- [ ] 응답 기록 보관 기간 결정 (보통 3~6개월)

> 광고성·마케팅성 발신은 정보통신망법 + 개인정보보호법 추가 검토 필요. 영업팀에 문의.

## 📊 발신 성공률 높이는 팁

| 팁 | 효과 |
|----|------|
| 발신 번호로 회사 대표번호 표시 (170 같은 휴대폰 번호 X) | 응답률 +20% |
| 오전 10-11시 또는 오후 2-4시에 발신 | 응답률 +15% |
| 멘트 첫 3초에 이름·소속 명시 | 끊김율 -30% |
| 시즈널 멘트 ("OO월 정기 안내입니다") | 친근감 ↑ |
| 1차 부재중 후 24시간 뒤 자동 재발신 1회만 | 최종 응답률 +25% |
