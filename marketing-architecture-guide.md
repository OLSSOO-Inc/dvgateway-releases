# Dynamic VoIP + DVGateway — AI 음성 서비스 플랫폼 아키텍처 가이드

> **OLSSOO Inc.** | AI-Ready Real-Time Voice Media Gateway

## 제품 개요

**Dynamic VoIP**는 기업용 클라우드 PBX이며, **DVGateway**는 실시간 전화 통화를 AI 파이프라인(STT·LLM·TTS)에 연결하는 고성능 미디어 게이트웨이입니다.

두 제품을 결합하면, 기존 전화 인프라를 변경하지 않고 **AI 음성 봇, 실시간 통역, 감정 분석, 자동 회의록** 등의 AI 서비스를 즉시 적용할 수 있습니다.

### 핵심 가치

| | 기존 콜센터 | Dynamic VoIP + DVGateway |
|---|---|---|
| **AI 연동** | 별도 개발 6-12개월 | SDK 연동 **1일** |
| **음성 봇** | 단순 ARS 메뉴 | 자연어 대화 AI (Claude, GPT) |
| **통화 분석** | 녹음 후 수동 확인 | **실시간** STT + 감정 분석 |
| **회의록** | 수기 작성 | 자동 생성 (JSON/TXT) |
| **비용** | 전용 장비 + 개발팀 | 클라우드 API 종량제 |

## SDK 코드 예시

### Python — 5줄로 AI 음성 봇 구축

```python
from dvgateway import DVGatewayClient
from dvgateway.adapters.stt import DeepgramAdapter
from dvgateway.adapters.llm import AnthropicAdapter
from dvgateway.adapters.tts import GeminiTtsAdapter

gw = DVGatewayClient(base_url="http://localhost:8080", auth={"type": "apiKey", "api_key": "dvgw_xxx"})
stt = DeepgramAdapter(api_key="dg_xxx", language="ko", model="nova-3")
llm = AnthropicAdapter(api_key="sk-ant-xxx", model="claude-sonnet-4-6",
                       system_prompt="친절한 AI 상담원입니다. 짧게 답변하세요.")
tts = GeminiTtsAdapter(api_key="AIza_xxx")

await gw.pipeline().stt(stt).llm(llm).tts(tts).start()
# 전화가 오면 AI가 자동 응대합니다
```

### 해피콜 봇 — 고객 맞춤 인사

```python
async def on_new_call(session):
    customer = session.custom_value_1 or "고객"
    order_id = session.custom_value_2 or ""
    greeting = f"안녕하세요 {customer}님, 주문번호 {order_id} 건에 대해 만족도 확인차 연락드렸습니다."
    await gw.say(session.linked_id, greeting, tts)

await gw.pipeline().stt(stt).llm(llm).tts(tts).on_new_call(on_new_call).start()
```

### 통화 제어 — AI 판단으로 통화 종료/전환

```python
async def on_new_call(session):
    await gw.say(session.linked_id, "무엇을 도와드릴까요?", tts)
```
