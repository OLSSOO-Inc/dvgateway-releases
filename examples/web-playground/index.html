    👋 DVGateway 체험 페이지에 오신 것을 환영해요

    여기는 전화 통화에 음성을 보내거나, 통화에서 들리는 음성을 처리하는 여러 기능을 직접 눌러보며 체험해 볼 수 있는 페이지예요.

    1. 왼쪽에서 서버 정보를 넣고 연결해요. (관리자에게 받은 주소·테넌트·비밀번호)
    2. 테스트용 전화를 한 통 걸어요. 게이트웨이가 통화를 인식하면 진행 중인 통화 목록에 나타나요.
    3. 왼쪽 데모 목록에서 마음에 드는 항목을 골라 클릭해요.
    4. 오른쪽 화면에 안내된 대로 버튼을 눌러 보세요. 결과가 통화에 즉시 반영돼요.

    처음에는 Callinfo 모니터 데모로 시작하시는 것을 추천해요. 전화 이벤트가 실시간으로 나타나는 모습만 봐도 게이트웨이가 어떻게 동작하는지 감이 잡혀요.
