    👋 DVGateway 체험 페이지에 오신 것을 환영해요

    여기는 전화 통화에 음성을 보내거나, 통화에서 들리는 음성을 처리하는 여러 기능을 직접 눌러보며 체험해 볼 수 있는 페이지예요.

    1. 왼쪽에서 서버 정보를 넣고 연결해요. (관리자에게 받은 주소·테넌트·비밀번호)
    2. 테스트용 전화를 한 통 걸어요. 게이트웨이가 통화를 인식하면 진행 중인 통화 목록에 나타나요.
    3. 왼쪽 데모 목록에서 마음에 드는 항목을 골라 클릭해요.
    4. 오른쪽 화면에 안내된 대로 버튼을 눌러 보세요. 결과가 통화에 즉시 반영돼요.

    처음에는 📞 통화 이벤트 살펴보기 데모로 시작하시는 것을 추천해요. 전화 이벤트가 실시간으로 나타나는 모습만 봐도 게이트웨이가 어떻게 동작하는지 감이 잡혀요.

    📖 용어 사전

    처음 보는 용어가 있어도 괜찮아요. 자주 나오는 단어들을 정리해 뒀어요.

    게이트웨이 (gateway)
    전화 시스템(PBX)과 인공지능 서비스를 연결해 주는 중간 서버예요. 통화 음성을 받아 인식·합성 서비스로 보내고, 결과를 다시 통화로 돌려보내는 역할을 해요.
    테넌트 (tenant) · 테넌트 ID
    한 게이트웨이를 여러 회사·팀이 나눠 쓸 수 있도록 만든 사용자 단위예요. 자기 테넌트 안의 통화·데이터만 보이고, 다른 테넌트와는 완전히 격리돼요.
    통화 ID (linkedId)
    한 통의 전화를 식별하는 고유한 번호예요. 보통 1779505908.318 같은 모양이에요. 통화에 음성을 보낼 때 어떤 통화인지 지정하는 데 써요.
    경량 모드 (mode=lite)
    가장 가볍게 동작하는 통화 모드예요. 미리 만들어 둔 안내음을 들려주고, 누른 키패드 번호를 받기만 해요. 음성 인식·합성 같은 무거운 기능은 안 써요. 통화 한 통당 자원 소모가 가장 적어요.
    풀스트림 모드 (mode=both, customer, agent)
    통화 음성을 실시간으로 게이트웨이에 보내고, 거기서 인식·합성·재생까지 자유롭게 처리하는 모드예요. 봇·실시간 자막·통역 같은 본격적인 기능에 써요.
    음성 합성 (TTS · Text-to-Speech)
    문자(텍스트)를 사람 목소리로 바꾸는 기술이에요. "안녕하세요"라고 적으면 그 문장을 자연스러운 음성으로 만들어 통화에 보내요. 데모에서는 ElevenLabs·OpenAI·Google Gemini 등을 골라 쓸 수 있어요.
    음성 인식 (STT · Speech-to-Text)
    사람 음성을 문자로 바꾸는 기술이에요. 회의에서 누가 말한 내용을 실시간 자막으로 보여주는 게 대표적인 예시예요.
    키패드 입력 (DTMF)
    전화기 키패드에서 0~9, *, # 같은 버튼을 누를 때 나는 신호예요. 자동응답에서 "1번 상담원 연결"처럼 메뉴를 선택할 때 써요.
    자동응답 (IVR · Interactive Voice Response)
    전화를 받아서 음성 안내를 들려주고, 키패드 입력에 따라 다른 안내로 넘어가는 시스템이에요. 은행 콜센터의 "주민번호 입력 후 # 버튼" 같은 게 IVR이에요.
    통화 이벤트 (callinfo)
    전화가 오고, 연결되고, 끊기는 매 순간을 알려 주는 신호예요. 데모는 이 이벤트를 받아서 화면에 표시하거나, 자동으로 인사말을 보내는 식으로 동작해요.
    방법 A · 방법 B (Mode A / Mode B)
    음성 합성·인식 키를 어디에 저장하느냐의 차이예요. 방법 A는 게이트웨이가 키를 받아서 직접 호출하고, 방법 B는 키를 서버에 보내지 않고 이 브라우저가 직접 호출해요. 보안상 권장은 방법 A예요.
    실시간 자막
    회의 통화에서 오가는 음성을 실시간으로 글자로 바꿔서 보여주는 기능이에요. 음성 인식(STT)을 이용해요.
    안내음 (sound:hello-world 같은 표기)
    전화 시스템에 미리 설치된 음성 파일을 지칭하는 표기예요. sound:hello-world는 "Hello, world" 안내음을 의미해요. digits:1234는 "1, 2, 3, 4"를 한 자리씩 읽어 주는 안내, tone:ring은 신호음이에요.
